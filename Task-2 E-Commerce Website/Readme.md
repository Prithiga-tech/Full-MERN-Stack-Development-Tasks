# NovaCart - Modern E-Commerce Website

Welcome to **NovaCart**, a modern and responsive e-commerce landing page built using HTML, CSS, JavaScript, and Bootstrap. This project is designed to deliver a futuristic online shopping experience with smooth animations, attractive UI, and interactive sections.

## Live Features

- 🏠 Hero / Landing Section with modern design and 3D-style effects  
- 🧾 About Us section describing the platform  
- ⭐ Why Choose Us section with key highlights  
- 🛍️ Services section showcasing product categories  
- 💬 Customer Testimonials with reviews  
- 📞 Contact Us form with submission success message using JavaScript  
- 📌 Footer with links and social media icons  
- 🎨 Smooth animations, hover effects, and transitions  
- 📱 Fully responsive design (mobile, tablet, desktop)

## Technologies Used

- HTML5  
- CSS3 (Custom Styling)  
- JavaScript (DOM manipulation)  
- Bootstrap 5  
- Font Awesome Icons  
- Google Fonts (Poppins)
