# Reunion Party Invitation Web Page

This is a simple and responsive **Reunion Party Invitation website** created using HTML, CSS. 
It displays event details in a modern and attractive design.

---

## Features
- Beautiful UI design with background image
- Glassmorphism style invitation card
- Event details (Date, Time, Venue)
- RSVP button with confirmation message
- Responsive design for mobile and desktop

---

## Technologies Used
- HTML
- CSS

---

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any browser
