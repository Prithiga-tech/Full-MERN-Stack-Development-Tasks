# MERN Stack Integration Project

## Overview
This project demonstrates the integration of a React.js frontend with a Node.js and Express.js backend using Axios API calls. Student information is stored in MongoDB and displayed dynamically on the frontend.

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- Axios

## Project Structure

MERN_Integration_Project
├── Server
│   ├── server.js
│   └── package.json
└── Client
    ├── src
    │   └── App.jsx
    └── package.json

## Features
- Add student names
- Store data in MongoDB
- Retrieve data using Express API
- Display student list in React
- Real-time frontend update using Axios

## Installation

### Backend
```bash
cd Server
npm install
node server.js
```

### Frontend
```bash
cd Client
npm install
npm run dev
```

## Output
- User enters a student name.
- Data is saved in MongoDB.
- Updated student list is displayed automatically.
